# Girish Panicker - Executive Resume Website

A premium single-page executive resume website designed to showcase leadership experience, operational excellence, customer assurance expertise, and enterprise-scale service management achievements.

## Overview

This website presents the professional profile of Girish Panicker, an experienced leader specializing in:

- Major Incident Management
- Customer Success & Escalation Governance
- Service Reliability Engineering (SRE)
- Enterprise Operations
- Change & Release Management
- AI-Enabled Process Transformation
- Global Support Operations

The site is designed as a modern digital resume with a luxury-inspired visual aesthetic, interactive navigation, and smooth user experience.

## Deployment

This project is a static website and can be hosted for free on:

- GitHub Pages
- Cloudflare Pages
- Netlify
- Vercel

### Local Execution

Simply open:

```bash
Girish_Panicker_Resume.html
```

in any modern browser.

Or run a local web server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
